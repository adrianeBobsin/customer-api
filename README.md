# Customer-api
## Description

RESTful API que suporta as seguintes operações:
● Salvar um cliente novo
● Atualizar um cliente existente
● Buscar um cliente por ID
● Autenticação

## Installation

```bash
$ npm install

$ docker pull redis

$ docker run -d -p 6379:6379 redis
```

## Running the app

```bash
# development
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
